Case 1: Analisis Data Buah
Saya menganalisis data buah menggunakan functional programming dan logika modular. Beberapa fitur yang saya terapkan:

Menampilkan semua nama buah secara unik.
Memisahkan buah berdasarkan tipe (LOCAL atau IMPORT) dan menghitung stok total masing-masing.
Memberikan komentar terkait data inconsistency, seperti duplikasi fruitId.
Case 2: Perhitungan Total Komentar
Saya menggunakan fungsi rekursi untuk menghitung total komentar, termasuk balasan (nested replies). Pendekatan ini memastikan fleksibilitas jika data komentar bertambah besar atau lebih kompleks.
